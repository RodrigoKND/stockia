# Stockia

**Stockia** is an inventory management platform designed for small businesses to save time on what matters most. It automatically registers products using AI and generates personalized digital catalogs based on user preferences.

Products can be shared through links and viewed as a public catalog, allowing businesses to showcase their inventory without hiring sales staff. Stockia acts as a simple digital seller, easy to configure and ready to use.

---

## Features

- Automatic product registration powered by AI
- Personalized product catalogs based on user preferences
- Shareable catalog links for customers
- Inventory-focused dashboard for business owners

## How It Works

1. Products are registered automatically using AI.
2. The system generates a personalized catalog based on user preferences.
3. Selected products can be published and shared via a public link.
4. Customers browse the catalog without accessing internal inventory data.
